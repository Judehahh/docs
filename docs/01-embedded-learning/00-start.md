# 入门

介绍一些嵌入式入门的知识。

## 写在前面

在学习一款 芯片/开发板 之前，需要有三个前提准备：
- 配置开发环境
- 准备开发板原理图
- 准备芯片手册

无论是进行以上准备工作，还是后续开展学习，都应擅用搜索引擎，在请教他人前先尝试在互联网上寻找答案，同时锻炼自己的信息检索能力。

> [!TIP]
> 尽量不要使用某度进行搜索，建议使用谷歌，没条件的话可以使用必应。
